# 実務特化型 プロンプト ＆ エージェント指示書ライブラリ (Prompt Library)

本ドキュメントは、日常業務および開発業務で即座に活用できる高精度プロンプトテンプレート（Template）とAntigravity用指示書のライブラリ集です。

---

## 1. 企画・文章作成・要約 (Writing & Summary)

### プロンプト1：長文ドキュメント・議事録の要約 (Summary Prompt)
```markdown
[Role]: 優秀なビジネスアナリスト
[Task]: 以下のテキストから、重要な決定事項、保留事項、ネクストアクション（担当者・期限）を抽出し、Markdownテーブル形式で要約してください。
[Format]:
## 1. 決定事項
## 2. 保留事項
## 3. ネクストアクション（テーブル）
[Input]:
{ここに文章を貼り付け}
```

### プロンプト2：プレスリリース・告知文の作成 (Writing Prompt)
```markdown
[Role]: Webマーケティングのプロライター
[Task]: 新サービス「{サービス名}」のプレスリリース記事を作成してください。読者の関心を引き、導入メリットが明確に伝わる構成にしてください。
```

---

## 2. 開発 ＆ コードレビュー (Coding & Review)

### プロンプト3：コードの安全なリファクタリング (Coding Prompt)
```markdown
[Role]: シニアソフトウェアエンジニア
[Task]: 以下のコードを、SOLID原則およびDRY原則に従ってリファクタリングしてください。パフォーマンスと可読性を向上させ、既存の動作を変えないでください。
[Constraint]: 変更理由の解説コメントをコード内に追加してください。
```

---

## 3. データ分析 ＆ リサーチ (Analysis & Research)

### プロンプト4：競合分析・市場調査 (Analysis Prompt)
```markdown
[Role]: 戦略コンサルタント
[Task]: {対象業界/テーマ}における主要な競合3社の強み・弱み・価格帯・ターゲット層を調査し、SWOT分析の形で整理してください。
```

---

## 4. Antigravity用 自動エージェント指示テンプレート (Agent Template)

```markdown
# エージェント行動指示書 (Template)
- あなたは {専門ロール} です。
- 計画（Planning）-> 実装（Implementation）-> テスト検証（Verification）のサイクルに従って自律実行してください。
- ファイルの削除や非互換なAPI変更を行う前に、必ず人間に計画を提示して承認を得てください。
```
