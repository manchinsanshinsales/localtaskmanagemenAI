# スマホ遠隔指示 ＆ デイリー自動運転セットアップガイド (Autonomous Mobile Setup)

本ガイドは、外出先や移動中に**「スマホ (Mobile) からエージェントへ仕事の指示を出し、自動（Autonomous）でタスクを実行・完了報告を受け取る」**ための遠隔運用セットアップ（Setup）マニュアルです。

---

## 1. 遠隔自律オペレーションの全体像

PCを開くことなく、スマホのGmailや通知画面から仕事を進めるアーキテクチャ：

```
[スマホ (Mobile)] ── (メール返信/指示) ──> [Gmail / Google Workspace]
                                                 │
                                                 ▼
[Antigravity / エージェント] <── (GAS API/通知) ── [GAS 自動連携]
        │
        ▼ (自律実行)
[成果物作成 ＆ テスト] ──> [スマホへ完了通知 (Mobile Report)]
```

---

## 2. スマホ遠隔指示のセットアップ手順

### ステップ1: Gmail自動通知機能の有効化
- `docs/scripts/gas_workspace_toolkit.js` を Google Apps Script に配置し、定時実行トリガーを設定します。

### ステップ2: スマホからの遠隔指示
- エージェントに依頼したいタスク（例：「経営層向けの週次レポートを作成して」「テストコードを追加して」）を、特定件名（例: `[AI-Task]`）で自分宛てのGmailに送信します。
- GASが件名を検知し、エージェント側へ指示をルーティングします。

### ステップ3: スマホでの完了確認
- タスクが完了すると、自動的にスマホのGmailに「完了報告（レスポンシブHTMLメール）」が届きます。添付またはリンクからスマホの縦画面で成果物を確認できます。
