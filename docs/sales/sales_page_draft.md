## 2. 販売文面 (Lander Text)

### 【このようなお悩みはありませんか？】
- 「生成AIを社内に導入したいが、機密情報漏洩やセキュリティルール作りに困っている」
- 「社員ごとにAIの使いこなし方に差があり、組織的な成果（ROI）が出ていない」
- 「AIエージェントにプログラミングや業務を自律実行させたいが、環境構築やプロンプトの型がわからない」

### 【本パッケージで手に入るもの】
1. **社内AIガバナンスガイドライン**: セキュアな運用のためのルールブック。
2. **実務特化型プロンプト＆指示書ライブラリ**: コピペで使える厳選プロンプト集。
3. **研修用スライド構成案**: 社内講義や勉強会ですぐに使えるプレゼンアセット。
4. **ROI評価フレームワーク**: 削減時間と投資効果を経営陣に可視化するツール。
5. **再現用自動化カスタムスキル (`ai-toolkit`)**: ワンコマンドで活用環境を自動構築。
